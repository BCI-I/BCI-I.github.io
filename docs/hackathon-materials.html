BCI initiative
Preparing for the hackathon

This page contains material to learn about the software and hardware that will be used for the BCI-I hackathon.


Accounts

In order to access some of the software needed, for example, to interact with the Duckiebots, you will need a GitHub account, a Docker Hub account and an account on the Duckietown Hub. Making sure that you have all accounts ready ahead of the competition will help you save time that you can spend hacking instead.

Hardware requirements

In order to participate in the hackathon, each time will require at least one computer that can run the software needed to interact with the EEG headset and the Duckiebots. The recomended platform is a MacBook with an Intel chip, computers Apple M1-2-3 chips may not work. All recomended software has been tested on this platform and should therefore run smoothly. Teams with laptop running Ubuntu should be able to get the software to work, but this has not been tested yet. Windows machine are not supported. Though it might be possible to get them to work using a virtual machine, this has not been tested and we encourage all teams to use a MacBook instead.


Streaming data from the EEG headset requires an Android device with Bluetooth connection. We strongly encourage all teams to ensure that they have access to such a device. We will have a very limited number of devices available for teams that do not have access to one, but we cannot guarantee that we will have enough for everyone.
